Student Location Tracking

This Location Tracking application is android based application used for tracking user location with the help of “Google API” for fetching the map data on our application and get geo location (Latitude , Longitude) of user by Location services of android devices. In this app user can able to see the location of other users of same App. User just need to know the Gmail id of other user to see the location of that particular user on map.

Technology used:-

  • Java programming language(OOP).
  • Android Studio with support API version of android 5 to 12..
  • Google Cloud Console , Google Map API, Google play services.
  • Firebase and XML.

Voice Assistant(Desktop Application)

Python provides an API called Speech Recognition to allow us to convert audio into text for further processing. The query for the assistant can be manipulated as per user's need. Speech recognition is the process of converting audio into text. This is commonly used in Alexa,Siri,etc.

Technology and software used:-

  • Python(Backend)
  • Python Speech Recognition(library).
  • Tkinter(Frontend).
  • Visual Studio Code
